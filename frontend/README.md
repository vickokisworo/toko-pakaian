# Frontend untuk Toko

Instruksi singkat menjalankan frontend development:

1. Buka terminal di folder `frontend`.
2. Jalankan:

```
npm install
npm run dev
```

Default API base diatur ke `http://localhost:3000/api`. Jika backend berjalan di host/port lain, buat file `.env` di folder `frontend` dan set `VITE_API_URL`.
